# Pacto — 画面ワイヤーフレーム v0.1

モバイル縦画面（PWA）を基準とした各画面のラフ。`[ ]` はボタン/操作要素。

## 画面遷移図

```
[ログイン]
   └─▶ [グループ選択] ──(新規作成)──▶ [グループ作成]
          │                └─(招待リンク)──▶ [グループ参加]
          ▼
      [商品一覧] ◀───────────────┐
          │  │  └─(＋)─▶ [商品登録/編集]
          │  └─(タブ)─▶ [お使いリスト一覧] ─▶ [お使いリスト詳細]
          ▼
      [商品詳細] ─(お使いに追加)─▶ お使いリストへ
```

下部タブ: **［商品］／［お使い］／［設定］**

## 5. 商品登録 / 編集

```
┌─────────────────────────┐
│ ✕ キャンセル      [保存]   │
├─────────────────────────┤
│ ┌─────┐ [📷 撮影/選択]    │
│ │[写真]│ ＋写真を追加       │  ← 複数枚（裏面・バーコードも）
│ └─────┘                  │
│ 商品名*  [______________] │
│ メーカー* [______________] │
│ カテゴリ  [洗剤        ▾]  │
│ 容量・規格[______________] │
│ 購入場所  [______________] │
│ メモ     [______________] │
│          [______________] │
└─────────────────────────┘
```
`*` は必須。写真は1枚以上必須。

## UI 方針メモ

- **写真ファースト**: 一覧も詳細もお使いリストも、まず写真が目に入るレイアウト。
- **片手操作**: 主要アクション（＋ / チェック）は親指の届く下部に配置。
- **オフライン耐性**: 一覧・写真はキャッシュ表示。店頭で電波が弱くても見られる。
- **最短2タップで撮影**: 一覧の「＋」→ カメラ起動を最短経路に。
